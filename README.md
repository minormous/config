Interfaces and classes for configuring [Auryn](https://github.com/rdlowrey/Auryn) in [Minormous](http://minormous.github.io/) projects.
Attempts to be [PSR-1](http://www.php-fig.org/psr/psr-1/), [PSR-2](http://www.php-fig.org/psr/psr-2/),
and [PSR-4](http://www.php-fig.org/psr/psr-4/) compliant.

For more information, see [the documentation](http://minormousframework.readthedocs.org/en/latest/#configuration).
